# Jogo de Adivinhação

Jogo onde você tem 5 tentativas para adivinhar um número secreto entre 1 e 50.

## Como Jogar

1. Execute o programa:
```bash
python adivinhacao.py
```

2. Digite um número entre 1 e 50

3. Use as dicas para ajustar seus palpites:
   - "Tente um número maior" → o número secreto é maior
   - "Tente um número menor" → o número secreto é menor

4. Acerte em até 5 tentativas para vencer!

## Exemplo

```
Digite o numero secreto de 1 a 50: 25
Oops! voce errou tente um numero maior

Digite o numero secreto de 1 a 50: 40
Oops! voce errou tente um numero menor 

Digite o numero secreto de 1 a 50: 35
parabens voce acertou
```

## Requisitos

- Python 3.x
